# luatos-lib-qqlbs

腾讯智能硬件定位(基站定位/wifi定位/蓝牙定位) for LuatOS

**非官方库,如有任意疑问或建议,请报issue,随缘更新**

## 介绍

本客户端基于socket库, 兼容所有LuatOS平台, 只要该平台实现socket库就行

## 安装

本协议库使用纯lua编写, 所以不需要编译, 直接将源码拷贝到项目即可

## 功能

1. 支持wifi定位
2. 支持基站定位
3. TODO 支持蓝牙定位, 但感觉没啥用?
